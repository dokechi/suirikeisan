Hydraulic Simple Calc MVP

水理計算シンプルMVP

給水装置の概算に必要な「同時使用水量 → 摩擦損失 → 所要水頭 → 口径比較」を、 ブラウザだけで一通り回せるようにした静的Webアプリです。

GitHub Pages 向け 依存関係なし オフライン利用可 最終設計前提ではない

今回のMVP範囲

まずは単線・給水装置寄りの簡易計算に絞っています。管網解析、火災時計算、ポンプや受水槽は次段階です。

基準の扱い

0.15 MPa / 0.74 MPa の判定、Weston と Hazen-Williams の切替、C=110 の既定値を内蔵しています。

使い方の流れ

まず同時使用水量を計算し、次に口径と延長から摩擦損失を計算し、その値を所要水頭・口径概算に流し込むのが基本です。

1. 同時使用水量

戸数式、人数式、標準化法、手入力に対応。

10戸未満: 42N^0.33 / 10戸以上600戸未満: 19N^0.67

30人以下: 26P^0.36 / 31〜200人: 13P^0.56

標準化法

13/20/25mm の標準使用水量を内蔵しています。13/20/25以外が多い場合は、下の直接入力を使ってください。

使用水量比表は 1〜30 器具まで内蔵しています。超える場合は将来拡張の対象です。

メモ

このMVPは、まず現場で判断の速い一次概算に寄せています。

  • 同時使用水量の根拠式は PDF の重複記載がある部分に寄せて実装しています。
  • 51〜74mm は資料記載の式切替の谷間なので、自動判定ではなく明示選択を推奨します。
  • メータ損失や器具必要水頭は事業体・機器条件で変わるので、ここは必ず現場条件で上書きしてください。
  • 将来的には受水槽・増圧・管網・帳票出力まで拡張できます。
