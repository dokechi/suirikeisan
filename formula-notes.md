# 公式メモ

このMVPで使っている式と、その典拠のページを整理したメモです。

## 1. 同時使用水量

### 1-1. 標準化法

**典拠:** `001856600.pdf` p.9 / `830003599.pdf` p.109

- 同時使用水量 = 給水用具の全使用水量 ÷ 給水用具総数 × 使用水量比
- 使用水量比表（内蔵値）
  - 1: 1.0
  - 2: 1.4
  - 3: 1.7
  - 4: 2.0
  - 5: 2.2
  - 6: 2.4
  - 7: 2.6
  - 8: 2.8
  - 9: 2.9
  - 10: 3.0
  - 15: 3.5
  - 20: 4.0
  - 30: 5.0
- 標準使用水量（内蔵値）
  - 13mm: 17 L/min
  - 20mm: 40 L/min
  - 25mm: 65 L/min

MVPでは 11〜14・16〜19・21〜29 は線形補間にしています。これは実装上の補完であり、最終設計時は元資料の扱いに合わせて確認してください。

### 1-2. 戸数式

**典拠:** `001856600.pdf` p.10 / `830003599.pdf` p.109

- 10戸未満: `Q = 42N^0.33`
- 10戸以上600戸未満: `Q = 19N^0.67`

### 1-3. 居住人数式

**典拠:** `001856600.pdf` p.10 / `830003599.pdf` p.109

- 30人以下: `Q = 26P^0.36`
- 31人以上200人以下: `Q = 13P^0.56`

`830003599.pdf` p.109 には 201 人以上の式や調査提案式もありますが、MVPでは両資料に共通している範囲に限定しました。

## 2. 摩擦損失水頭

### 2-1. Weston 公式

**典拠:** `001856600.pdf` p.13

口径 50mm 以下で使用。

- `h = (0.0126 + (0.01739 - 0.1087D) / V^0.5) * (L / D) * V^2 / 2g`
- `Q = πD^2 / 4 * V`

ここで:

- `h`: 摩擦損失水頭 (m)
- `V`: 平均流速 (m/s)
- `L`: 管長 (m)
- `D`: 管径 (m)
- `g`: 重力加速度
- `Q`: 流量 (m³/s)

### 2-2. Hazen-Williams 公式

**典拠:** `001856600.pdf` p.14 / `830003599.pdf` p.69

口径 75mm 以上で使用。

- `h = 10.666 * C^-1.85 * D^-4.87 * Q^1.85 * L`

MVPでは流速は補助的に次式で表示しています。

- `V = Q / (πD^2 / 4)`

### 2-3. C値

**典拠:** `001856600.pdf` p.14 / `830003599.pdf` p.69

- 新管で屈曲部損失などを含んだ管路全体: `C = 110`
- 直線部のみ: `C = 130`

MVPの既定値は `110` です。

## 3. 圧力基準

**典拠:** `830003599.pdf` p.67

- 分岐箇所の最小動水圧: `0.15 MPa 以上`
- 分岐箇所の最大静水圧: `0.74 MPa 以下`

## 4. 参考表示

**典拠:** `001856600.pdf` p.13

- 管内流速について「2.0 m/s 以下としている」という参考記述があるため、MVPでは警告表示だけ入れています。
- これは本ツールでは**参考指標**であり、絶対判定には使っていません。
